## 原文摘录
> 第一，在 Claude Code 这类 Coding Agent 里，上下文不是一句两句对话，而是一个大杂烩：系统提示词、长期记忆、调用过的工具、工具的输出、读过的代码文件、终端日志、以及你给它的各种指令，统统都会塞进去。模型每回答一次，都是在这一大坨信息里“分配注意力”，决定该看什么、不看什么。
